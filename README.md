# DoD Housing Automation Framework

## Mission Description
Automated testing suite for the Military Housing enterprise system (34 modules).
This project ensures 2.5 million service members have reliable access to housing.

## Technical Architecture
* **Language:** Python 3.x
* **Web Automation:** Selenium WebDriver
* **Database:** SQLite (Oracle Simulation)
* **CI/CD:** GitHub Actions

## Developer Setup
1.  Clone repository.
2.  Run tests using `pytest`.
